## 🚀 What is SYNC.?

SYNC. is a proximity-based, offline-first social network.

- Connects people within 50m  
- No internet required  
- No likes, no followers  
- Messages disappear instantly

## 🔥 Core System

- 📡 Proximity (Bluetooth + GPS)  
- 🚫 Zero Numbers  
- ⏳ Ephemeral Communication  
- 🛡 AI Moderation
